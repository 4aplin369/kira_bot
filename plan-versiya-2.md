### Коллажи из фото 🖼

Собираем коллажи из фото животика. Нужны **все четыре формата**:

- **Лента прогресса** — сетка из всех фото по неделям (от меньшего животика
  к большему).
- **Сравнение 2 фото** — две выбранные недели рядом, «было → стало».
- **По месяцам** — отдельный коллаж за каждый месяц беременности.
- **Финальный к родам** — один большой красивый коллаж со всей беременностью.

**Триггеры — оба:**

- **По кнопке** — подруга сама собирает нужный формат, когда захочет.
- **Авто раз в месяц** — бот сам присылает обновлённую ленту прогресса
  (через `JobQueue`, по аналогии с утренним сообщением).

**Технический подход:**

- Сборка изображений — библиотека **Pillow (PIL)** (добавить в `requirements.txt`).
- В базе хранится Telegram `file_id`; для коллажа бот **скачивает** файлы
  (`bot.get_file(file_id)` → байты/файл) — для композиции нужны пиксели,
  а не `file_id`.
- Нормализация: фото разной ориентации/размера приводим к единой ячейке
  (масштаб + центрированная обрезка), подписи недель — шрифтом PIL.
- Объёмы 20–40 фото — по памяти и времени некритично.
- Нужен шрифт для подписей: положить `.ttf` в репозиторий или взять системный.
