# 🤰 Бот для беременной подруги — Версия 1

Тёплый бот-сюрприз. Эта версия **ничего не хранит** — её можно спокойно дарить, терять нечего. Дневник и фото добавим во второй версии, когда ты сама проверишь хранение.

## Что внутри

- `main.py` — код бота
- `content.py` — все тексты (приветствия, недели, «можно/нельзя», чек-листы)
- `requirements.txt` — что нужно установить

## Что бот умеет

- 🌅 **Доброе утро** — 30 тёплых сообщений, выбираются случайно
- 👶 **Размер малыша** — по текущей неделе, с фруктом/овощем, недели можно листать
- ⏳ **Обратный отсчёт** — сколько дней до встречи
- 🔍 **Можно / нельзя** — еда, кофе, лекарства, спорт
- ✅ **Чек-листы** — вопросы врачу, анализы, дела по триместрам
- 🦶 **Шевеления** — счётчик в рамках одного захода

Дата ПДР сейчас стоит **15.10.2026** (ориентировочная). Бот сам считает неделю — сегодня выходит 22-я.

## Запуск за 5 шагов

### 1. Создай бота в Telegram
Открой в Telegram **@BotFather** → команда `/newbot` → придумай имя и юзернейм → он пришлёт **токен** (длинная строка вида `12345:AAterm...`). Сохрани его.

### 2. Установи Python
Скачай с [python.org](https://www.python.org/downloads/) (версия 3.10 или новее). При установке на Windows поставь галочку **«Add Python to PATH»**.

### 3. Положи файлы в одну папку
`main.py`, `content.py`, `requirements.txt` — все вместе.

### 4. Установи зависимости
Открой терминал (или «Командную строку») в этой папке и выполни:
```
pip install -r requirements.txt
```

### 5. Впиши токен и запусти
Самый простой способ — открой `main.py` в Блокноте, найди строку:
```python
BOT_TOKEN = os.environ.get("BOT_TOKEN", "СЮДА_ВСТАВЬ_ТОКЕН_ОТ_BOTFATHER")
```
и замени `СЮДА_ВСТАВЬ_ТОКЕН_ОТ_BOTFATHER` на свой токен (в кавычках). Сохрани.

Запусти бота:
```
python main.py
```
Готово! Открой своего бота в Telegram, нажми **Start** — он оживёт.

> Пока запущен `python main.py`, бот работает. Закроешь терминал — бот «уснёт». Для теста этого достаточно.

## Как поправить дату ПДР
Когда узнаешь точную дату, открой `main.py`, найди:
```python
DUE_DATE = date(2026, 10, 15)
```
и поставь нужную (год, месяц, день). Перезапусти — всё пересчитается само.

## Чтобы бот работал круглосуточно
Чтобы он не «засыпал», когда твой компьютер выключен, его ставят на бесплатный хостинг (например, Railway, Render, PythonAnywhere — у них есть бесплатные тарифы). Это следующий шаг — когда дойдём до второй версии с дневником, я распишу хостинг подробно: там как раз подключим базу и облако для фото.
